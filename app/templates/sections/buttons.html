Buttons

Add the @mixin button($color) to create any color button with the basic button styles. Button styles are not applied to the <button> tag directly. For standard buttons, use .button, which has @mixin button($tribune-yellow) included.

Show @mixin button($color)
      
      @mixin button($color) {
          @include clearfix;
          background-color: $color;
          border-bottom: .25em solid darken($color, 4);
          border-radius: 3px;
          color: $bodytext-black;
          display: inline-block;
          font-family: $base-sans-serif;
          font-weight: 600;
          letter-spacing: .05rem;
          line-height: 1em;
          margin: 0 0 .25em;
          padding: .5em .75em .625em;
          text-align: center;
          vertical-align: middle;

          &:active,
          &:hover {
              background-color: transparentize($color, .3);
              border-bottom: .125em solid darken($color, 6);
              margin-top: .125em;
          }
      }

      .button {
        @include button($tribune-yellow);
      }
