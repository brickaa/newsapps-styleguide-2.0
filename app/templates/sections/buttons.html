Buttons

Add the @mixin button($color) to create any color button with the basic button styles. Button styles are not applied to the <button> tag directly. For standard buttons, use .button, which has @mixin button($tribune-yellow) included. For a ghost button, add the argument 'ghost': @mixin button($color, ghost). The class .ghost-button will include a teal ghost button.

      
        @mixin button($color, $type: null) {
            @include clearfix;
            cursor: pointer;
            display: inline-block;
            font-family: $base-sans-serif;
            letter-spacing: .03rem;
            margin: 0 0 0.3em;
            text-align: center;
            vertical-align: middle;

            @if $type == ghost {
                background: transparent;
                border: 2px solid $color;
                color: $color;
                text-transform: uppercase;
                padding: 0.5em 0.75em;

            } @else {
                background: $color;

                @if $color == $teal {
                    color: $white;

                } @else {
                    color: $bodytext-black;
                }
                border: 0px;
                padding: .5em .75em .625em;
            }

            &:active,
            &:hover {
                @if $type == ghost {
                    background-color: transparentize($color, 0.95);

                } @else {
                    background-color: transparentize($color, .3);
                }
            }
        }

        .button {
            @include button($tribune-yellow);
        }

        .ghost-button {
            @include button($teal, ghost);
        }
