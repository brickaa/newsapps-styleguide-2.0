Prose

Basic prose styles, and the follow components, can be found in theme/_prose.scss.

To add our basic paragraph styles to a class, @extend %prose. You can also wrap elements in .prose to apply the basic paragraph styles.

Show %prose & .prose
      
        /* Base Prose Theme */
        %prose {
          @extend %font-base;
          color: $bodytext-black;
          display: block;
          font-family: $base-serif;
          font-weight: 400;
          line-height: 1.4;
          text-align: left;
        }

        // Prose Settings
        .prose {
          @extend %prose;
          @include clearfix;

          ul {
            @extend %font-s;
            font-family: $base-sans-serif;
            font-weight: 600;
            list-style: disc;
            padding: 0 1em;
          }

          li {
            margin-bottom: .5em;

            &:last-child {
              margin-bottom: 1em;
            }
          }

          img {
            width: 100%;
          }

        }

Headlines, Bylines

Use the classes .headline and .byline for basic headlines.

Cat Saves Planet from Apocolypse, Dies in Action

By Captain Planet | July 0, 2084

Show HTML
      
      <header>
        <h1 class="headline">Cat Saves Planet from Apocolypse, Dies in Action</h1>
        <p class="byline">By <a href="http://www.texastribune.org/about/staff/alexa-ura/">Captain Planet</a> | July 0, 2084</p>
      </header>
Show CSS
      
        .headline {
          @extend %font-l;
          font-family: $base-sans-serif;
          font-weight: 600;
          margin: 1.25em 0 .6em;
        }

        .byline {
          @extend %font-s;
          font-family: $base-sans-serif;
          margin-bottom: 0;

          a {
            font-weight: 800;
          }
        }

Captions

Captions for photos and other media elements are styled using .caption. Images within .prose will fill 100% of the available width, unless otherwise specified.

This is a picture of a kitten inside a div.prose.

Show HTML
      
        <div class="prose">
          <img src="http://placekitten.com/800/400">
          <p class="caption">This is a picture of a kitten inside a div.prose. </p>
        </div>
Show CSS
      
        .caption {
          @extend %font-s;
          color: $text-gray;
          font-family: $base-sans-serif;
          font-weight: 400;
          margin: .25em 0;
        }

Paragraph Headers

That ^ is an example of a .paragraph-header. Notice it will automatically transform the text to uppercase. You can also use these styles on other elements using @extend %paragraph-header.

Show HTML
      
        <h3 class="paragraph-header">Paragraph Headers</h3>
Show CSS
      
        %paragraph-header,
        .paragraph-header {
          font-family: $base-sans-serif;
          font-size: .875em;
          font-weight: 700;
          letter-spacing: 1px;
          margin: 1em 0;
          text-transform: uppercase;
        }

Explainer Text

Use .explainer-text when you're explaining something, such as how to use the Hospitals Explorer or to define a glossary definition. You can also @extend %explainer-text to another class.

Show HTML
      
      <p class="explainer-text">Text goes here</p>
Show CSS
      
        %explainer-text,
        .explainer-text {
          background: $explainer-gray;
          color: $bodytext-black;
          font-family: $base-sans-serif;
          font-weight: 300;
          line-height: 1.3;
          padding: 1em;
        }

Prose Blocks

Add .prose-block around elements that you want to highlight inside your prose with a yellow border on top and bottom, such as block-quotes or embedded lists. This class does not need to be wrapped inside .prose.

Example list

  • Lists are often used to call out important points.
  • They typically begin with an .paragraph-header.
  • Exciting stuff.
Show List HTML
      
      <div class="prose-block">
        <h3 class="paragraph-header">Example list</h3>
        <ul class="bullet">
          <li>One</li>
          <li>Two</li>
          <li>Three</li>
        </ul>
      </div>
Show .prose-block
      
        .prose-block {
          border-color: $tribune-yellow;
          border-style: solid;
          border-width: 1px 0;
          margin-bottom: calc(16rem / $base);
          position: relative;
        }

Block Quotes

Use the class .block-quote for the text of a block quote, and .block-quote-source for the attribution. Also, make sure to use HTML special characters for your quote marks! They won't be styled correctly if you forget. **Warning: The example code below is converting the special characters, so make sure to change it if you copy/paste.**

“You know what’s awesome? Cats. Cats are awesome.”

— Becca Aaronson, cat enthusiast

Show HTML
      
      <div class="prose-block">
        <h2 class="block-quote">“You know what’s awesome? Cats. Cats are awesome.”</h2>
        <p class="block-quote-source">— Becca Aaronson, cat enthusiast</p>
      </div>
Show .block-quote .block-quote-source
      
      .block-quote {
        @extend %font-m;
        font-family: $base-serif;
        font-style: italic;
        font-weight: 700;
        line-height: 1.2em;
        margin: 1em .5em;
        position: relative;
        text-align: left;
      }

      .block-quote-source {
        @extend %font-base;
        font-family: $base-sans-serif;
        font-weight: 300;
        margin: 1em .5em;
      }
Select & Input

The classes .prose and .graphic include base styles for <select> and <input> fields. If you add these elements outside of .prose or .graphic, they will not have any styles applied.
